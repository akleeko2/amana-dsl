# Amana App Suite

This folder freezes the language surface for a set of full Amana applications.
The apps here should only use implemented language/runtime features.

Current status:

1. `01-ops-horizon` - built and verified first.
2. `02-customer-care-hub` - planned.
3. `03-finance-suite` - planned.
4. `04-team-pulse` - planned.
5. `05-analytics-lab` - planned.

Rules:

- No language surface expansion while building these apps.
- Every runtime bug found here becomes a regression test.
- Docs for each app live beside the app source.
